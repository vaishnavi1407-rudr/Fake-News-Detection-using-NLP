# Fake News Detection using NLP

This project detects whether a news article is fake or real using Natural Language Processing and Machine Learning.

## Technologies Used
- Python
- NLTK
- Scikit-learn
- TF-IDF
- Logistic Regression

## Steps in the Project
1. Load Fake and Real news datasets
2. Text preprocessing (tokenization, stopword removal)
3. TF-IDF vectorization
4. Train/Test split
5. Logistic Regression model training
6. Prediction of fake or real news

## Model Accuracy
~98%
